Platform Sync Guide

Windows / macOS / Steam: easiest path. Enable Detailed Logs in MTGA, restart MTGA, open Collection, then run Import MTGA Collection.

iOS: export logs from device to Mac/PC, then run import with those paths.

Android: easiest reliable path is to log into the same account on desktop (shared account), then import from desktop logs.

Online decklists: paste list into Optimize Imported Decklist to get collection-aware swap priorities.
